# 键盘与鼠标

键盘和鼠标是你与电脑对话的最主要工具。它们每天承受着数万次敲击和滑动，却也最容易被忽视——直到某个按键突然失灵、无线鼠标开始乱跳，或是不知怎么触发了奇怪的输入模式。

这一节，我们准备了：

- 使用指南：如何正确连接有线/无线键鼠、更换电池、调整指针速度和双击节奏，以及那些能让你效率倍增的键盘快捷键。
- 常见故障：无线键鼠突然没反应、鼠标指针乱跳或卡顿、个别按键打不出字、笔记本自带键盘误触、外接键盘与自带键盘按键冲突等问题的完整排查步骤。

无论你用的是 Windows 还是 Mac，我们都会分别给出对应路径。请根据手边的设备对号入座。

## 使用指南

### 如何连接有线/无线键盘与鼠标

#### Windows

##### 有线键鼠：

- 将 USB 插头直接插入电脑的 USB 接口即可。系统会自动识别并安装基础驱动，等待几秒后即可使用。如果是老式 PS/2 接口（圆形插头），需在关机状态下插拔，开机后才能识别。

##### 无线键鼠（带 USB 接收器）：

1. 将无线接收器插入电脑 USB 口（推荐使用前置接口或附赠的延长底座，让接收器远离金属机箱和 USB 3.0 接口）。
2. 打开鼠标/键盘底部的电源开关。
3. 大部分免驱型号插入即可使用。如果需要额外对码，请参考设备说明书，通常是按住底部的 Connect 键 3~5 秒。

##### 蓝牙键鼠：

1. 打开鼠标/键盘的电源开关，并长按蓝牙配对键（通常是一个蓝牙图标按钮），直到指示灯开始闪烁。
2. 电脑端：进入“设置 → 蓝牙和其他设备 → 添加设备 → 蓝牙”。
3. 在列表中点击找到的设备名称，等待配对完成。

#### Mac OS

##### 有线键鼠：

- 将 USB 插头直接插入 Mac 的 USB-A 或通过转接头/USB-C Hub 连接。系统会自动识别，首次连接时可能会弹出“键盘设置助理”，按提示按几个键让系统识别按键布局即可。
- 无线键鼠（带 USB 接收器）：插入接收器后，进入“系统设置 → 蓝牙”，确认蓝牙已开启。部分 2.4G 无线键鼠不需要蓝牙，插入即可用。

##### 蓝牙键鼠：

1. 打开鼠标/键盘电源开关，长按蓝牙配对键直到指示灯闪烁。
2. Mac 端：进入“系统设置 → 蓝牙”，确保蓝牙已开启。
3. 在“附近的设备”列表中找到你的键盘或鼠标，点击“连接”。
4. 如果配对时需要输入验证码，在键盘上输入屏幕显示的六位数字后按回车确认。

##### Apple 原厂设备（Magic Keyboard / Magic Mouse / Magic Trackpad）：

- 用自带的 USB-C 转 Lightning 或用 USB-C 线缆连接到 Mac，一次有线连接即可自动完成配对。之后拔掉线缆即可无线使用。

### 如何调整鼠标指针速度与双击速度

#### Windows

1. 进入“设置 → 蓝牙和其他设备 → 鼠标”。
2. 点击右侧的“其他鼠标设置”，会弹出一个经典设置窗口。
3. 调整指针速度：在“指针选项”标签页，拖动“移动”栏下的滑块——往左更慢，往右更快。建议不要勾选“提高指针精确度”，这会让指针产生加速度感，不利于稳定操控。
4. 调整双击速度：在“鼠标键”标签页，拖动“双击速度”滑块。双击右边的文件夹图标测试手感：如果文件夹能正常打开/关闭，说明速度合适。
5. 在这个窗口中，你还可以调整滚轮一次滚动的行数、指针轨迹等选项。

#### Mac OS

1. 进入“系统设置 → 鼠标”。
2. 调整跟踪速度（指针移动速度）：拖动“跟踪速度”滑块，往左更慢，往右更快。
3. 调整双击速度与连按速度：
    - 拖动“连按速度”滑块，旁边没有测试区域，你可以找一个文件夹双击测试手感，调到你顺手为止。

4. 调整滚动方向：
    - 默认开启“自然滚动”（手指向上滑，页面内容向上滚，类似触屏手机的逻辑，也叫“反向滚动”）。
    - 如果你更习惯 Windows 的传统向下滚轮逻辑，请关闭“自然滚动”。

5. Magic Mouse 更多手势：在“鼠标”设置中，还可以为右键、轻点、滑页等手势配置操作。
6. 如果使用的是非 Apple 鼠标：部分功能可能需安装厂商提供的驱动软件（如 Logitech Options）才能调节侧键等功能。

## 常见故障

### 无线鼠标/键盘突然失灵

#### Windows

1. 更换电池：这是无线键鼠失灵的最常见原因。请先换上一颗确保有电的新电池。
2. 检查电源开关：很多无线鼠标底部或侧面有一个物理电源开关，可能在携带时误拨到了“Off”。确认它处于“On”位置。
3. 换个 USB 口插入接收器：
    - 建议换到电脑前侧的 USB 口，或使用附赠的 USB 延长线让接收器远离机箱。
    - 不要将无线接收器插在 USB 3.0 接口（蓝色）旁边的相邻口，USB 3.0 的电磁干扰会严重影响 2.4G 无线信号。

4. 重新对码：用细针或笔尖，按住鼠标/键盘底部的 Connect 或配对孔 3~5 秒，直到指示灯闪烁或系统弹出设备就绪提示。
5. 在设备管理器中检查：右键“开始”→“设备管理器”→“鼠标和其他指针设备”和“键盘”。如果设备上有黄色感叹号，右键“卸载设备”，然后拔插接收器或重启电脑，让系统重新识别。

#### Mac OS

1. 更换电池或充电：同上，电量不足是最常见的原因。检查鼠标/键盘是否有低电量指示灯闪烁。
2. 检查电源开关：确认设备电源开关处于“On”位置。
3. 检查蓝牙连接：进入“系统设置 → 蓝牙”，查看设备是否还在“我的设备”列表中，状态是否为“已连接”。如果显示“未连接”，点击设备名尝试重新连接。
4. 重新配对：在“蓝牙”设置中，点击设备名旁边的 `ⓘ` 信息图标 → “忽略此设备”。然后重新按配对流程添加一次（参考本节“如何连接”部分）。
5. 移除可能的无线干扰：将手机、无线充电器等移离键鼠和 Mac；避免在 Mac 与键鼠之间放置金属物品。
6. 重置 Mac 蓝牙模块（仅 Intel 芯片 Mac 推荐）：按住 `Shift + Option`，点击菜单栏蓝牙图标 → “还原蓝牙模块” → 好。此操作会断开所有蓝牙设备，需重新配对。Apple Silicon 芯片 Mac 直接重启即可重置蓝牙状态。

### 鼠标指针乱跳或卡顿

#### Windows

1. 清洁底部光学透镜：用棉签或无绒布轻轻擦拭鼠标底部的发光孔区域。积灰和细小纤维会干扰光学传感。
2. 换一张合适的鼠标垫：避免在玻璃、亮面反光桌面、透明材质上直接使用光学鼠标。推荐使用单色、无图案、表面细腻的鼠标垫。
3. 排除无线干扰：
    - 把无线接收器从 USB 3.0 接口旁边移开。
    - 将手机、无线充电器等电磁设备移远一些。
    - 使用接收器延长底座，让接收器直接放到桌面上，靠近鼠标。

4. 调整指针速度：进入“设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标设置 → 指针选项”，取消勾选“提高指针精确度”后测试。

#### Mac OS

1. 清洁底部光学透镜：同上，用棉签轻擦清洁。
2. 检查追踪表面：Magic Mouse 对透明、反光表面较敏感。换一张不反光的单色鼠标垫测试。
3. 检查蓝牙干扰：
    - 如果同时连接着多个蓝牙设备（耳机、键盘、触控板），尝试暂时断开其他设备，只留鼠标，看是否改善。
    - 移除 Mac 与鼠标之间的金属杂物。

4. USB 3.0 / Thunderbolt 设备干扰：部分外接硬盘或扩展坞会产生 2.4GHz 频段干扰。尝试暂时断开这些设备，看鼠标是否恢复正常。
5. 调整跟踪速度：进入“系统设置 → 鼠标”，重新调整“跟踪速度”到顺手值。

### 键盘某个按键打不出字

#### Windows

1. 先排除简单的碎屑卡键：
    - 把键盘倒过来，轻轻拍打背面，让食物碎屑、灰尘掉出来。
    - 用压缩空气罐或吹气球对准失灵按键的缝隙吹气。

2. 测试是轴体问题还是键帽问题（台式机键盘）：可以小心把失灵的键帽取下来，直接按压下面的轴体。如果轴体能正常触发，只是键帽没装好或损坏——问题不大，清洁重装或换键帽即可。
3. 外接键盘测试（笔记本用户）：插一个外接键盘，如果外接键盘的相同按键能正常打出，那说明笔记本自带键盘物理损坏，需要报修。如果外接键盘的这个按键也打不出，可能是系统输入法或按键映射问题。
4. 检查筛选键和粘滞键：“设置 → 辅助功能 → 键盘”，确保“筛选键”和“粘滞键”都已关闭。这些辅助功能在误开后会改变键盘的触发行为。

#### Mac OS

1. 先排除碎屑卡键：同样，将键盘倾斜轻轻拍打，或用压缩空气吹扫缝隙。
2. 外接键盘测试（MacBook 用户）：插一个外接键盘，如果外接键盘同位置按键能打出，说明自带键盘物理损坏，需报修。
3. 检查按键映射：进入“系统设置 → 键盘 → 键盘快捷键”，确认没有不小心为那个按键分配了奇怪的功能。可以在“修饰键”中检查是否误将某个按键映射为其他功能（如把 A 键映射成了 Control）。
4. 检查慢速键和粘滞键：进入“系统设置 → 辅助功能 → 键盘”，关闭“慢速键”和“粘滞键”测试。
5. 重启 Mac：一次重启能解决很多临时的按键映射错乱问题。

### 笔记本自带键盘误触关闭方法

有时外接了键盘后，笔记本自带的键盘在打字时容易被手掌误触，或者只想用外接键盘，想把自带的给关掉

#### Windows

- 方法一（最彻底）：右键“开始”→“设备管理器”→“键盘”，找到“内置键盘”（名称中通常有 “Standard PS/2 Keyboard” 或制造商名如 “HID Keyboard Device”），右键选择“禁用设备”。此操作后笔记本键盘立刻失效，只有外接键盘能用。⚠️ 注意：重启后登录界面如需输入密码，禁用自带键盘后可能无法输入，务必确保外接键盘插着并能工作。
- 方法二（品牌自带工具）：部分品牌笔记本（如 ThinkPad、戴尔、惠普）在出厂系统中自带键盘管理工具或 Fn 快捷键组合，可以在键盘驱动或厂商控制软件中临时禁用内置键盘。

#### Mac OS

- macOS 没有内置一键禁用自带键盘的选项，外接键盘时，MacBook 自带键盘默认仍可工作。
- 物理阻隔方案：你可以使用一些第三方工具（如 Karabiner‑Elements）来在连接外接键盘时自动禁用内置键盘，但这需要安装额外软件且配置较复杂。如果不熟悉，建议来联系 IT 部门协助设置。
- 使用 iPad 式的“随航”或外接屏幕方案：如果你习惯合盖使用外接显示器、外接键鼠的方式，直接合上 MacBook 盖子即可（须接电源适配器且外接显示器亮着）。这样自带键盘自然不会误触。

### 外接键盘与自带键盘按键冲突

有时你会发现外接键盘和笔记本自带键盘同时存在时，按下某些键会出现奇怪的现象（比如按一次出现两个字符，或者按键反应错乱）。

#### Windows

1. 输入法冲突是主因，而非硬件冲突：首先尝试按 `Win + 空格` 切换输入法，或者在任务栏语言栏点击确认当前的输入语言是“中文”还是“英文”。多数“按键冲突”现象其实是输入法的中英文状态不是你期望的。
2. 在设备管理器中禁用其中一个键盘：如果你真的只需要外接键盘工作，可以按上一小节的方法，在设备管理器中禁用内置键盘。这样就不会有任何“冲突”可能。
3. 检查是否误开了筛选键/粘滞键：设置 → 辅助功能 → 键盘，关闭所有已开启的辅助功能。

#### Mac OS

1. 检查输入法：点击菜单栏的输入法图标，确认当前输入源是“ABC”还是“拼音”等。Mac 也能连接两个键盘，它们共享当前输入法状态，一般不会冲突。
2. 检查键盘类型识别：Mac 连接外接键盘时，会弹出“键盘设置助理”，要求你按几个键来识别布局。如果这次识别错误，可能造成某些键映射不对。进入“系统设置 → 键盘 → 键盘快捷键 → 修饰键”，检查两个键盘的修饰键映射是否正确。
3. 一次只连接一个外接键盘以减少变量。如果按键依旧异常，可能是外接键盘本身布局导致的，尽量使用标准布局的外接键盘。
