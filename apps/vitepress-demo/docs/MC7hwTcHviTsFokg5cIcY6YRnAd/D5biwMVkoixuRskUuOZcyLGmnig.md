# 显示器

显示器是你每天面对时间最长的办公设备。一个设置得当的屏幕能让眼睛更舒适，工作更高效；而连接不上、画面闪烁、比例失真这些小问题，又常常在最不合适的时刻冒出来。

这一节，我们准备了：

- 使用指南：如何正确连接显示器、切换扩展/镜像模式、调整亮度和色温，以及多屏排列的设置技巧。
- 常见故障：外接无画面、黑屏、闪烁条纹、比例失真、笔记本合盖后外接不亮等问题的排查步骤。

Windows 和 Mac 在显示器管理上各有逻辑，我们分别给出了对应路径。请根据你使用的设备对号入座。

## 使用指南

### 如何连接显示器并设置扩展/复制模式

#### Windows

1. 使用 HDMI、DP 或 VGA 线将电脑与显示器连接好，并确保显示器已通电开机。
2. 按下快捷键 `Win + P`，屏幕右侧会弹出投影模式菜单。
3. 四种模式任你选：
    - 仅电脑屏幕：外接显示器不工作，只用笔记本或主机自带屏幕。
    - 复制：两个屏幕显示完全相同的内容，适合演示 PPT。
    - 扩展：两个屏幕拼成一个大桌面，可以把窗口在两个屏幕间自由拖动——这是最推荐的工作模式。
    - 仅第二屏幕：关闭自带屏幕，只使用外接显示器。

4. 用方向键或鼠标点选你需要的模式，即刻生效。

#### macOS

1. 使用合适的线缆（或转接头/扩展坞）将 Mac 与外接显示器连接，并确保显示器已通电开机。
2. 进入“系统设置” → “显示器”。
3. 系统通常会自动检测到外接显示器，并默认以“扩展”模式工作。
4. 如果要切换模式：
    - 在“显示器”设置中找到“用作”下拉菜单。
    - 选择“扩展显示器”：两个屏幕各自显示不同内容（推荐工作模式）。
    - 选择“镜像显示器”：两个屏幕显示完全相同内容（适合演示）。

5. 如果连接后外接显示器没有画面，请参考本节“常见故障”中的排查步骤。
6. 📌 重要提示：首次连接新显示器时，macOS 可能会弹出“允许连接配件”的提示，请选择“允许”。

### 如何调整显示器亮度与色温

#### Windows

##### 调整亮度：

1. 笔记本：通常使用键盘上的亮度调节快捷键（F键上带太阳图标），或者点击任务栏右下角的通知中心图标，在亮度滑块上拖动。
2. 台式机/外接显示器：一般无法在系统内直接调亮度，建议使用显示器自带的物理按钮进入 OSD 菜单调节。

##### 夜间模式（降低蓝光）：

1. 进入“设置 → 系统 → 屏幕 → 夜间模式”。
2. 可以手动开启，也可以设定为“日落到日出”自动切换。
3. 拖动色温滑块，建议设在 4800K–5000K 左右，屏幕会偏暖偏黄，长时间工作更护眼。

#### Mac OS

##### 调整亮度：

1. 使用键盘上的亮度调节键（通常为 F1 和 F2，带太阳图标）。
2. 也可以进入“系统设置 → 显示器”，拖动“亮度”滑块。

##### 自动调节亮度：

1. MacBook 默认会根据环境光自动调节屏幕亮度。如果想手动控制，可在“系统设置 → 显示器”中关闭“自动调节亮度”。

##### 夜览（Night Shift，降低蓝光）：

1. 进入“系统设置 → 显示器 → 夜览”。
2. 可以手动开启到明天，或按“日落到日出”定时自动开启。
3. 拖动色温滑块调整暖度，建议偏向“更暖”一端，对双眼更友好。

##### 原彩显示（True Tone）：

1. 在“系统设置 → 显示器”中，可选择开启或关闭“原彩显示”。开启后会根据环境光自动调节屏幕色彩和亮度，观感更自然。如果您从事对颜色要求精确的工作（如设计、修图），建议关闭该功能。

### 如何设置多显示器排列顺序

当你同时使用笔记本自带屏幕和外接显示器时，可能会发现鼠标从屏幕边缘穿过去的方向不太对——这就需要调整显示器的“逻辑排列”了。

#### Windows

1. 右键桌面空白处 → “显示设置”。
2. 你会看到两个（或更多）标着数字的矩形框，分别代表你的每一块屏幕。
3. 用鼠标按住并拖拽这些矩形框，让它们的相对位置和你桌面上的物理摆放完全一致。例如：
    - 外接显示器放在笔记本左侧并垫高了，就把矩形 2 拖到矩形 1 的左边。
    - 外接显示器是竖屏，就点选它的矩形，在下方“显示方向”中选择“纵向”。

4. 点击“应用”。设置好后，鼠标就能顺畅地在屏幕之间穿行了。
5. 如果不确定哪个矩形对应哪块屏幕，可以点击“标识”按钮，每块屏幕上都会短暂显示对应的数字编号。

#### Mac OS

1. 进入“系统设置 → 显示器”。
2. 点击“排列”按钮（会显示两个代表屏幕的矩形图示）。
3. 用鼠标拖拽这些矩形，让它们的相对位置和桌面上的物理摆放一致。例如：
    - 外接显示器实际放在左侧，就把它的矩形拖到内置屏幕矩形的左边。
    - 白色菜单栏条所在的那块屏幕是“主屏幕”。想更换主屏幕，直接用鼠标把白色菜单栏条拖到另一块屏幕的矩形上即可。

4. 设置好后，鼠标跨屏移动就会顺畅自然。
5. 如果你不确定哪块矩形对应哪个屏幕，在任意矩形上按住鼠标，其对应屏幕的边缘会显示红色边框提示。

## 常见故障

### 显示器无画面 / 黑屏

#### Windows

1. 确认显示器是否上电：检查显示器电源指示灯是否亮起。如果不亮，检查电源线是否插紧、插座是否有电、显示器电源开关（通常在底部或背面）是否打到“开”。
2. 确认投影模式：连续按几次 `Win + P`，选择“复制”或“扩展”。误选“仅电脑屏幕”是外接显示器黑屏的最常见原因。
3. 确认信号源：使用显示器边框上的物理按钮，进入 OSD 菜单，找到“输入源”（Input Source），手动切换到你实际连接的那个接口（如 HDMI1、HDMI2、DP、VGA）。信号源不匹配是极常见的问题。
4. 检查视频线材：确认视频线两端都已插紧。如果线材外观有明显弯折、破损，换一根好线测试。
5. 跳过中间设备：如果使用了 Type-C 扩展坞、USB 转 VGA/HDMI 转接头等中间设备，尝试把显示器的视频线直接插到电脑自带的视频口上（如果你的电脑有相同接口）。很多“无画面”问题其实是中间转接器的兼容问题。
6. 笔记本外接测试：用另一台笔记本连接同一台显示器。如果别的电脑能点亮，问题在你的电脑；如果也点不亮，问题在显示器或线材。

##### Mac OS

1. 确认显示器是否上电：同上，检查显示器电源指示灯。不亮则检查电源线、插座、显示器电源开关。
2. 检查系统是否识别到外接显示器：
    - 进入“系统设置 → 显示器”。
    - 如果系统没能识别到外接显示器，这里只显示内置屏幕的设置项。
    - 如果识别到了，会在设置中显示两个显示器的图标。点击外接显示器的图标，确认“用作”已选为“扩展显示器”或“镜像显示器”。

3. 确认信号源：使用显示器物理按钮进入 OSD 菜单，确认信号源与实际连接的接口一致。
4. 检查线材和扩展坞：重新插拔视频线两端；如果用了转接头或扩展坞，尝试用合适的线缆直连 Mac 的 Thunderbolt/USB-C 端口。
5. MacBook 合盖后外接不亮：参考本节“笔记本合盖后外接显示器无信号”的排查方法。
6. 另一个设备测试：用另一台电脑连接同一台显示器。如果别的电脑能点亮，问题在你的 Mac；如果也点不亮，问题在显示器或线材。

### 显示器画面闪烁或出现条纹

#### Windows

1. 检查刷新率：进入“设置 → 系统 → 屏幕 → 高级显示 → 显示器 1 的显示适配器属性 → 监视器”标签页，在“屏幕刷新频率”下拉框里，务必选择数值最高的那一项（如 60Hz、75Hz、144Hz 等）。错误或偏低的刷新率会直接导致闪屏。
2. 检查线缆连接：重新插拔显示器视频线的两端。轻轻晃动线缆，观察闪烁是否跟着加剧——如果是，线缆可能内部断裂，换一根好线测试。
3. 排除干扰源：将手机、微波炉、大功率充电器等电磁干扰源移离显示器和视频线。
4. 判断是主机还是显示器问题：如果只是外接显示器闪，笔记本自带屏幕不闪，问题大概率在视频线、转接头或显示器本身。如果两个屏幕都闪，可能是显卡驱动或系统问题。

#### Mac OS

1. 检查刷新率：进入“系统设置 → 显示器”，确认“刷新率”已设为显示器所支持的最高值（如 60Hz、120Hz 或更高）。部分显示器连接 Mac 后默认刷新率可能偏低。
2. 检查线缆与扩展坞：重新插拔视频线；如果用了转接头或扩展坞，尝试直连 Mac。Mac 对某些第三方扩展坞的兼容性可能不够理想，这是闪屏的常见原因。
3. 仅外接闪屏而 Mac 自带屏幕正常：问题大概率是外接显示器、视频线或转接头的问题，可以逐个替换测试。
4. 仅 MacBook 自带屏幕闪屏：进入“系统设置 → 显示器”，关闭“自动调节亮度”测试。如果问题持续，可能涉及环境光感应器或屏幕硬件，需报 IT。

### 外接显示器画面比例失真 / 显示不全

#### Windows

1. 使用显示器自带菜单（OSD）调整：按下显示器边框上的物理按钮，进入 OSD 菜单。找到“画面比例”、“图像比例”或类似选项，设置为“全屏”或“Fill to Screen”。不要选“1:1”或“点对点”。
2. 调整显卡控制面板：右键桌面 → 显示更多选项 → 你的显卡控制面板（NVIDIA / AMD / Intel）：
    - NVIDIA：显示 → 调整桌面尺寸和位置 → 缩放模式 → 选“全屏”，并勾选“覆盖游戏和程序设置的缩放模式”。
    - AMD：显示器 → 缩放模式 → 选“全面板”。
    - Intel：显示器 → 缩放 → 选“全屏”。

3. 检查分辨率设置：确认系统“显示设置”中，外接显示器的分辨率已选择“推荐”选项。

#### Mac OS

1. 检查分辨率：进入“系统设置 → 显示器”，确认已选择“默认”模式。如果选了“缩放”，按住 `Option` 键点击某个缩放选项，展开详细分辨率列表，逐个尝试找到完整显示的那一个。
2. 使用显示器自带菜单（OSD）调整：同样，使用显示器物理按钮进入菜单，把“画面比例”或“宽高比”设为“全屏”或“自动”。
3. 部分第三方显示器存在 EDID 兼容问题：如果以上方法都无效，显示器可能向 Mac 传了错误的显示信息。可以尝试使用 BetterDisplay 等工具强制覆盖分辨率，或联系 IT 协助。

### 笔记本合盖后外接显示器无信号

#### Windows

当你把笔记本盖子合上，希望只使用外接显示器工作时，默认的电源选项可能会让电脑进入“睡眠”状态，导致外接也黑屏。

1. 打开“控制面板” → “硬件和声音” → “电源选项”。
2. 在左侧菜单中，点击“选择关闭计算机盖的功能”。
3. 在“关闭盖子时”这一栏，把“用电池”和“接通电源”两种情况都改为 “不采取任何操作”。
4. 点击“保存修改”。
5. 修改后，合上盖子，电脑会继续工作，但可能需要手动把显示模式切成“仅第二屏幕”（`Win + P`）以获得最佳效果。

#### Mac OS

MacBook 合盖使用外接显示器（也叫“盒盖模式”或“闭合模式”）需要满足以下条件：

1. 必须连接电源适配器：MacBook 在电池供电状态下合盖会自动进入睡眠，必须插上充电器才能合盖持续工作。
2. 必须连接外接键盘和鼠标/触控板：合盖后自带的键盘和触控板无法使用，要提前备好外接键鼠并确保它们已连接、可操作。
3. 必须外接显示器已连接并亮起：先连好外接显示器并确保有画面，然后再合盖。
4. 如果上述条件都满足但合盖后外接仍无信号：
    - 进入“系统设置 → 锁定屏幕”，检查相关设置。
    - 尝试“系统设置 → 电池 → 选项”，确保未开启“如果可能，在关闭显示器时使用电池供电”之类选项。
    - 重启 Mac 后再试一次。

5. 如果问题持续，可能是外接显示器或扩展坞唤醒信号兼容性问题，请尝试更换连接方式或联系 IT。
