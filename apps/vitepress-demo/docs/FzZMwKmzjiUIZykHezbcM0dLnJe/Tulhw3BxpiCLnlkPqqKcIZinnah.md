# 安装系统

## 安装教程

## 常见问题

### 使用启动 U 盘安装系统时，扫描不到本地磁盘

#### 方法一：加载 RST/VMD 驱动

- <b>准备驱动</b>：在另一台能联网的电脑上，去对应电脑/主板品牌的官网，搜索型号并下载芯片组分类里的“Intel Rapid Storage Technology （IRST） （英特尔快速存储技术）”驱动。
- <b>解压文件</b>：下载后解压，把包含很多文件（尤其是`.inf`等）的完整文件夹，整个复制到你的系统安装 U 盘里。
- <b>加载驱动</b>：用 U 盘启动，在“找不到驱动器”界面，点击 “加载驱动程序”，再点击“浏览”，找到并选中你 U 盘里的驱动文件夹，点击“确定”。系统会自动搜索并加载，成功后硬盘就会出现了。

::: tip 

- 很多时候找不到驱动，是因为只把驱动文件（如。exe 安装程序）单独放在了 U 盘，需要确保先在别的电脑上“解压”出所有文件，复制整个文件夹。

:::

#### 方法二：在 BIOS 里调整 SATA 模式

这个方法对所有平台都适用，操作就是把“RAID”改成“AHCI”（无需加载额外驱动）。

- 适用症状：BIOS 里能看到硬盘，但 Windows 安装程序里看不到。
- 操作方法：重启电脑按`F2`或`Del`进入 BIOS，找到 “SATA Operation” 或 “SATA Mode”，把设置从 “RAID On” 改成 “AHCI”，然后`F10`保存退出。

`AHCI`和`RAID`（或带有 RST）的区别是：前者兼容性最好，装系统基本不用额外驱动；后者开启后，BIOS 里的硬盘信息可能会消失，但如果要使用多块硬盘组阵列或部分傲腾加速功能，装系统时就必须手动加载驱动。

::: tip 注意

改动 BIOS 的硬盘模式，有极小概率会影响现有系统盘启动。如果硬盘只有一块且用来装系统，一般影响不大。若存在其他重要系统盘，请谨慎操作。

:::

> 参看文档：https://www.asus.com.cn/support/faq/1044458/#A2

### 安装 Windows 11 跳过联网

#### 方法一：使用 `OOBE\BYPASSNRO` 命令

- <b>步骤</b>：在“让我们为你连接到网络”界面，按 `Shift + F10` 打开命令提示符，输入 `OOBE\BYPASSNRO` 后按回车。
- <b>特点</b>：系统会自动重启，之后界面底部会增加“我没有 Internet 连接”的选项。方法相对成熟，在多数版本的安装镜像上都有效。

#### 方法二：使用 `start ms-cxh:localonly` 命令 （更快捷）

- <b>步骤</b>：同上，打开命令提示符后，输入 `start ms-cxh:localonly` 后按回车。
- <b>特点</b>：无需重启电脑，可直接创建本地账户并完成后续设置。操作更简便，可以优先尝试。

#### 方法三：使用 Rufus 工具 （一劳永逸）

- 步骤：在另一台电脑上，用最新版 Rufus 写入 Windows 11 ISO 镜像时，在“自定义 Windows 安装”选项中，勾选“移除对在线 Microsoft 账户的要求”。
- 特点：用制作好的安装 U 盘启动并安装系统时，可直接创建本地账户，无需再执行任何命令。
