# 电脑与系统

本章是针对电脑主机和操作系统（Windows 或者 Mac OS） 的核心操作与常见问题，涵盖如何查看配置、调整屏幕、安装卸载软件等基础指南，以及无法开机、蓝屏死机、系统卡顿、显示异常、弹窗广告、无声等常见故障的详细排查步骤。

##### 查看系统版本与构建号

按 `Win + R`，输入 `winver` 并回车，弹出窗口会显示详细版本号（如 Windows 11 专业版 22H2）。

##### 查看核心硬件配置摘要

1. 打开“任务管理器” ：
    1. 在任务栏空白处右键，选择“任务管理器”。
    2. 使用快捷键： `Ctrl + Shift + Esc`。

2. 切换到“性能”选项卡，可分别看到：
    1. CPU：型号、基准速度、逻辑处理器数量。
    2. 内存：总量、已用/可用，若“已用”长期接近100%，是卡顿的典型原因。
    3. 磁盘：型号与容量，关键看“活动时间”，若持续100%则说明硬盘高负荷运作。
    4. 网卡：实时收发速率，可初步判断网络是否拥堵。

#### Mac OS

##### 查看系统版本与硬件概况

点击屏幕左上角的苹果图标  → “关于本机”。这里会直接显示 macOS 版本名称（如 macOS Sequoia 15.2）、芯片型号（如 Apple M3 或 Intel）和内存大小。

##### 查看更详细的硬件信息

在“关于本机”窗口，点击“更多信息”或“系统报告”，可以分别查看：

- 储存空间：各磁盘分区的已用/可用容量。
- 内存：总量及当前使用压力。
- 网络：当前连接的网络接口和 IP 地址。

### 如何设置屏幕缩放与分辨率

#### Windows 

- 右键桌面空白处 → “显示设置”。
- 缩放：系统会自动标出“推荐”比例。如果你觉得图标和文字偏小，可以试着调到 125% 或 150%。注意：个别旧软件在改缩放后会发虚，可参考本章“字体模糊”的解决方法。
- 分辨率：强烈建议始终选择带“推荐”字样的分辨率。这是屏幕物理像素和系统输出“点对点”的最佳状态，最清晰。随意降低会导致整个画面变朦胧。

#### Mac OS

- 进入“系统设置” → “显示器”。
- 缩放：默认是“默认”，系统自动使用最优分辨率。如果你希望看到更多内容（文字和窗口变小），可以选择“更多空间”；反之选择“更大字体”。
- 分辨率：macOS 通常不直接显示具体分辨率数字，而是用缩放选项来管理。系统默认选择就是当前屏幕的最佳显示效果。如果外接显示器画面异常，请在“显示器”设置中，按住 `Option` 键，点击“缩放”来展开详细的分辨率列表进行手动选择。

### 安装软件

为了电脑安全和清爽，安装软件请遵循以下原则。Windows 和 Mac 在软件来源和安全机制上有所不同，需分别留意。

#### Windows 

1. 从正确渠道下载：在下载软件安装包前一点要去软件对应的官网下载。绝对不要用第三方高速下载站或捆绑安装器。
2. 安装过程保持警觉：
    - 向导页面中，注意取消勾选“安装 XX 安全卫士”“设置 XX 导航为首页”等附加项。
    - 当出现“选择安装位置”时，点击“浏览”把盘符从 C 改成 D（除非软件强制要求 C 盘）。
    - 杀毒软件弹出“是否允许此应用对您的设备进行更改？”时，确认是你主动安装的再点“是”。

3. 牢记红线：❌ 未经公司批准，禁止安装任何破解版、注册机、盗版商业软件和游戏。

#### Mac OS

1. 从正确渠道获取：
    - 优先使用公司指定的方式：如自有的软件分发系统、公司提供的安装包。
    - App Store：打开 App Store 应用，搜索并下载已上架的工作软件（如 Microsoft 365、企业微信、钉钉等）。App Store 中的软件经过 Apple 安全审核。
    - 官网下载：对于不在 App Store 的软件（如 Chrome 浏览器），务必去官网下载 `.dmg` 安装包。

2. 安装方式：
    - 下载 `.dmg` 文件后双击打开，在弹出窗口中，将应用图标拖拽到右侧的“应用程序”文件夹即可完成安装。
    - 部分软件是 `.pkg` 安装包，双击后按向导提示完成。

3. 如果提示“无法打开，因为它来自身份不明的开发者”：这是 macOS 的安全机制。请不要随意关闭。如果确认是正常的工作软件，打开“系统设置”→“隐私与安全性”，在页面底部会有该软件的放行按钮，点击“仍然打开”。不建议在设置中全局关闭 Gatekeeper。
4. 牢记红线：❌ 未经公司批准，禁止安装任何破解版、注册机、盗版商业软件和游戏。

### 卸载软件

#### Windows 

- 推荐方式：打开“设置” → “应用” → “已安装的应用”。搜索目标软件名，点右侧三个点 → “卸载”，按提示完成。
- 备选方式：打开“控制面板” → “程序和功能”，右键目标程序 → “卸载”。
- 📌 有些软件卸载后仍会留下弹窗广告，可联系 IT 用专业工具帮你扫尾。

#### Mac OS

- 标准方式：打开“访达”（Finder）→“应用程序”文件夹，找到要卸载的软件，直接拖进废纸篓，然后清空废纸篓。
- 彻底清理：部分软件会留下偏好设置和缓存文件。可使用 App Cleaner 等免费工具辅助彻底卸载，或联系 IT 协助。
- 📌 如果软件自带卸载程序，请运行软件文件夹中的卸载器，而非直接拖入废纸篓。

### 查看电脑 IP 地址与 MAC 地址

#### Windows 

1. 按 `Win + R`，输入 `cmd` 并回车，打开命令提示符。
2. 输入 `ipconfig /all` 后回车。
3. 找到你正在用的网络适配器：
    - 插着网线：找“以太网适配器 以太网”。
    - 连着 Wi‑Fi：找“无线局域网适配器 WLAN”。

4. IPv4 地址：形如 `192.168.x.x` 或 `10.x.x.x`。若为 `169.254.x.x`，说明没能正常获取 IP，本身就是一种网络故障。
5. 物理地址（MAC）：一串十六进制字符，如 `00-1A-2B-3C-4D-5E`。

#### Mac OS

1. 打开“系统设置” → “网络”。
2. 选择当前正在使用的网络接口（Wi‑Fi 或以太网），点击“详细信息”。
3. 在右侧信息面板中：
    - IP 地址：会直接显示形如 `192.168.x.x` 的 IP 地址。
    - MAC 地址（硬件地址）：显示为“Wi‑Fi 地址”或“以太网地址”，形如 `00:1A:2B:3C:4D:5E`（用冒号分隔）。

4. 你也可以在终端应用中输入 `ifconfig` 并回车，效果类似 Windows 下的 ipconfig。

### 设置系统自动锁屏时间

养成让电脑在你离开时自动锁屏的习惯，是保护信息安全最简单有效的方式。

#### Windows 

- 进入“设置 → 个性化 → 锁屏界面”，往下滑，点击“屏幕超时设置”。
- 可以分别对“用电池”和“接通电源”设定关闭屏幕和进入睡眠的时间。
- 建议将下方“需要登录”设为“当电脑从睡眠中唤醒时”，确保息屏后任何人点亮它都必须输入密码。

#### Mac OS

- 进入“系统设置” → “锁定屏幕”。
- 可以分别设置：
    - “使用电池供电时在不活跃后关闭显示器”的时间。
    - “使用电源适配器时在不活跃后关闭显示器”的时间。
    - “关闭显示器后需要密码”：建议设为“立即”，确保一旦息屏就必须输入密码或 Touch ID 才能解锁。

## 常见故障

### 无法开机 / 黑屏

#### Windows 

1. 从供电查起：
    - 笔记本：拔掉电源适配器，仅靠电池开机。能亮说明适配器坏了；仍不亮就插回适配器，长按电源键 30 秒以上释放主板静电，再短按开机。
    - 台式机：检查机箱电源线是否松动、插排开关是否被误关。观察主机电源风扇或主板指示灯是否通电。

2. 排除外设短路：拔掉所有 USB 设备（只留键鼠），再按开机键。
3. 判断是主机还是屏幕问题：若电源灯亮、风扇转但屏幕无显示：
    - 笔记本：按亮度增加键排除误调亮度；连接外接显示器测试。若外接有画面，说明笔记本屏幕或排线故障，需报 IT。
    - 台式机：检查显示器电源和视频线是否插紧，用显示器按钮手动切换到正确的输入源。

#### Mac OS

1. 从供电查起：
    - MacBook：拔掉充电器，仅靠电池开机。能亮说明充电器或线材故障；不能亮则插回充电器，长按 `Touch ID / 电源键` 10 秒以上强制关机，再按一次正常开机。
    - iMac / Mac mini / Mac Studio：确认电源线两端都已插紧、插座有电。观察设备是否有指示灯亮起。

2. 排除外设短路：拔掉所有 USB / Thunderbolt 设备（只保留键鼠），再按开机键。
3. 判断屏幕问题（仅 MacBook）：
    - 按键盘上的亮度增加键（通常为 F2），排除亮度被误调至最低。
    - 连接外接显示器测试。若外接有画面但自带屏无显示，说明屏幕或排线硬件故障，需报 IT。

### 登录界面输入密码后一直转圈

#### Windows 

- 首选办法：断网登录。 拔掉网线或关闭 Wi‑Fi，再输入密码。域账户在断网时会使用本地缓存凭据验证，通常能立刻进入桌面。登录后再重新连接网络，联系 IT 检查账户同步状态。
- 若断网后仍卡住超过 5 分钟，长按电源键强制关机，等待 10 秒后重启。

#### Mac OS

- 强制重启再试：长按 `Touch ID / 电源键` 约 10 秒强制关机。等待几秒后重新开机。
- 重置 NVRAM（仅 Intel 芯片 Mac）：开机时按住 `Option + Command + P + R` 键约 20 秒，听到两次启动声后松开。Apple Silicon 芯片 Mac 会自动检查 NVRAM 状态，一般无需手动操作。
- 使用安全模式启动：开机时按住 `Shift` 键，直到看到登录窗口。如果能成功登录，说明是某些开机启动项或扩展导致的问题。正常重启退出安全模式后，清理不必要的启动项。
- 若反复无法登录，可能是账户或系统问题，请联系我们。

### 电脑运行越来越卡、反应慢

顿不是突然出现的，是后台慢慢堆积出来的。从以下几个核心点入手，通常能找回流畅。

#### Windows 

1. 清理开机自启程序：`Ctrl + Shift + Esc` 打开任务管理器 →“启动”标签页，右键禁用非必要的、且“启动影响”标为“高”的程序。
2. 释放 C 盘空间：若“此电脑”中 C 盘变红，进入“设置 → 系统 → 存储”，开启“存储感知”并立即运行。关键操作：将微信/企业微信聊天记录保存路径从 C 盘迁移到 D 盘。
3. 检查磁盘占用：任务管理器“进程”页按“磁盘”列排序。
4. 强烈建议把应用安装在非C盘。

#### Mac OS

1. 检查登录项与后台进程：进入“系统设置”→“通用”→“登录项与扩展”，检查“登录时打开”列表，移除不需要随开机启动的程序。
2. 释放磁盘空间：进入“系统设置”→“通用”→“储存空间”。系统会分类显示占用情况，按建议清理：
    - 清空废纸篓。
    - 删除不再需要的 iPhone/iPad 备份、旧下载文件。
    - 打开“优化 Mac 储存空间”，将不常用的文件自动转存到 iCloud。

3. 检查内存压力与资源占用：打开“活动监视器”（在“应用程序/实用工具”中，或按 `Command + 空格` 搜索）。
    - 切换到“内存”标签页，底部有“内存压力”图。如果长期显示为红色，说明内存不够用。
    - 切换到“CPU”标签页，按“% CPU”排序，找到占用异常高的进程，选中它，点左上角“×”强制退出。

4. 定期重启：macOS 虽然稳定，但每周主动重启一次能清除很多临时文件和内存占用。

### 死机 / 蓝屏 / 内核崩溃

#### Windows 

- 你最该做的第一件事：用手机拍下屏幕底部带下划线的终止代码（如 `DRIVER_IRQL_NOT_LESS_OR_EQUAL`）。这是给 IT 的最准确线索。大部分蓝屏重启后可暂时恢复。

常见代码及应对：

- `DRIVER_IRQL_NOT_LESS_OR_EQUAL`：驱动冲突。重启时反复按 F8 进入安全模式，在设备管理器中回退或卸载最近更新的驱动。
- `CRITICAL_PROCESS_DIED`：系统进程崩溃。以管理员身份运行命令提示符，执行 `sfc /scannow` 修复系统文件。
- `INACCESSIBLE_BOOT_DEVICE`：启动盘无法访问。⚠️ 高风险，有数据丢失可能。立即停止操作，报 IT。
- `PAGE_FAULT_IN_NONPAGED_AREA`：内存或虚拟内存问题。关机后重新插拔内存条（需 IT 协助），或在高级系统设置中重置虚拟内存。

#### Mac OS

- 应用无响应（鼠标能动，单个程序卡住）：
    1. 按 `Command + Option + Esc`，调出“强制退出应用程序”窗口。
    2. 选中显示为“（无响应）”的程序，点击“强制退出”。

- 整个系统冻结（鼠标也不能动了）：
    1. 长按 `Touch ID / 电源键` 约 10 秒，直到屏幕变黑，强制关机。
    2. 等待几秒后，再按一次电源键正常开机。

- 内核崩溃（屏幕变暗，出现多国语言提示“你的电脑因出现问题而重新启动”）：
    - 这是 macOS 级别的严重错误，等同于 Windows 的蓝屏。通常会自动重启。
    - 重启后，如果再次出现，请用手机拍下屏幕上的具体错误文本，并留意：
        - 出现崩溃前你正在使用什么软件或外设？
        - 是否刚刚插入了某个新硬件（如外置网卡、扩展坞）？
    - 将这些信息和截图一起报给 IT 部门。

### 外接显示器无画面 / 显示不全

#### Windows 

1. 先按切换键：反复按几次 `Win + P`，选择“扩展”。误选“仅电脑屏幕”是外接黑屏的最常见原因。
2. 确认信号源：用显示器按钮进入 OSD 菜单，找到“输入源”，手动切成你实际连接的那种接口（如 HDMI 2）。
3. 跳过转接头测试：如果你用了扩展坞或转接头，尝试把视频线直接插到电脑本体的视频口。
4. 画面比例失真：在显示器 OSD 菜单中，把“画面比例”设为“全屏”或“Fill to Screen”。

#### Mac OS

1. 检查连接与识别：进入“系统设置”→“显示器”。确定 macOS 是否检测到了外接显示器（会显示两个显示器的图标）。如果没有，检查线缆连接、尝试重新插拔或换一根线缆。
2. 切换显示模式：
    - 在“显示器”设置中，找到“用作”下拉菜单，选择“扩展显示器”（两个屏幕显示不同内容）或“镜像显示器”（两屏显示相同内容）。
    - 可以通过拖拽显示器图标来调整它们的相对位置，让鼠标跨屏移动顺畅。

3. 使用排列检测：如果显示器支持，点击“排列”按钮，macOS 会尝试自动识别显示器位置。
4. 跳过扩展坞测试：如果使用了 Type‑C 扩展坞，尝试将显示器直接连接 Mac 的 Thunderbolt 端口（通过合适的线缆或转接头）。
5. 画面比例异常：在“显示器”设置中，按住 `Option` 键点击“缩放”，可展开详细分辨率列表，逐个尝试找到合适的选项。

### 字体模糊 / 图标异常变大或变小

#### Windows 

- 个别老软件模糊：右键其快捷方式 → 属性 → 兼容性 → 更改高 DPI 设置 → 勾选“替代高 DPI 缩放行为”，下拉选择“应用程序”。
- 全局模糊：确认显示设置中缩放比例为“推荐”值，并更新显卡驱动。

#### Mac OS

- macOS 的高分屏（Retina）适配非常成熟，很少出现全局字体模糊。
- 外接显示器文字发虚：通常是因为分辨率没设对。去“系统设置”→“显示器”，确认分辨率已选择“默认”。如果是非 Retina 外接显示器，按住 `Option` 键点击“缩放”，逐个尝试找到最清晰的那个选项。
- 个别网页或 App 字体细小：按 `Command + +` 快捷键可以放大内容，`Command + 0` 恢复默认大小。

### 屏幕闪烁 / 出现条纹

#### Windows 

1. 刷新率先查：设置 → 系统 → 屏幕 → 高级显示 → 监视器，将刷新率设为最高值。
2. 检查连接线缆：重新插拔视频线，外观有弯折破损则更换测试。
3. 查看电源是否有异响。

#### Mac OS

1. 检查刷新率：进入“系统设置”→“显示器”，确认“刷新率”已设为显示器支持的最高值（如 60Hz、120Hz 或更高）。
2. 检查线缆与扩展坞：尝试重新插拔视频线、更换线缆或跳过扩展坞直连测试。
3. 仅外接闪屏而 Mac 自带屏幕正常：问题大概率出在外接显示器、视频线或转接头上。
4. 仅 MacBook 自带屏幕闪屏：可能是环境光感应器误判，关闭“系统设置”→“显示器”中的“自动调节亮度”测试；若问题持续，可能涉及硬件。

### 关机时强制更新、等待时间过长

#### Windows

- 严禁在更新界面长按电源键断电，可能导致系统文件损坏。
- 合理规避：利用午休或会议间隙，主动去“设置 → Windows 更新”安装补丁，避免下班时集中强制更新。

##### Mac OS

- macOS 的系统更新通常会给用户选择“立即更新”或“稍后提醒”，很少在工作结束时强制更新。
- 如果遇到更新耗时较长：请确保电源适配器已连接（MacBook），不要强制关闭，耐心等待。为避免打断工作，建议在午休或下班后手动去“系统设置”→“通用”→“软件更新”主动完成。
- ❌ 严禁在更新进度条走动时强制断电或合盖（若未接电源），可能导致系统损坏。

### 电脑频繁弹窗广告 / 右下角广告推送

#### Windows

1. 精准定位广告源头：弹窗出现时，`Ctrl + Shift + Esc` 打开任务管理器 → 进程，找到可疑条目 → 右键“打开文件所在位置”，直接定位并卸载。
2. 清理网站通知：Edge → 设置 → Cookie 和网站权限 → 通知 → 拉黑来路不明的网站。

##### Mac OS

macOS 系统本身极少受到传统的桌面弹窗广告困扰，更多是浏览器层面的通知骚扰。

1. 检查浏览器通知：
    - Safari：打开 Safari → 菜单栏“Safari”→“设置”→“网站”→“通知”，把来路不明的网站“拒绝”。
    - Chrome：设置 → 隐私与安全性 → 网站设置 → 通知，拉黑不需要的网站。

2. 检查可疑软件：如果经常收到不正常的广告弹窗，可能安装了恶意软件。打开“访达”→“应用程序”文件夹，检查是否有不认识的可疑软件，将其卸载。或使用 Malwarebytes 等工具扫描一次。
3. 一般来说，只要你只从 App Store 或官网下载软件，macOS 不会遇到这类问题。

### 系统声音消失 / 喇叭显示红叉

#### Windows

1. 右下角喇叭图标有红叉时：右键开始 → 设备管理器 → 声音、视频和游戏控制器。
2. 若有黑色向下箭头，右键“启用设备”；若有黄色叹号，右键“卸载设备”（勾选删除驱动），重启电脑。

##### Mac OS

1. 最基础检查：点击菜单栏右侧的声音图标（喇叭），检查是否被误调至静音或音量拉到最低。按键盘上的音量增加键（通常为 F12）测试。
2. 检查输出设备：
    - 按住 `Option` 键，同时点击菜单栏的声音图标，可以看到“输出设备”列表。
    - 确保选中的是“MacBook Pro 扬声器”或“内置扬声器”，而不是其他设备（如误选为某个外接显示器的音频输出）。

3. 核心音频重置：打开“活动监视器”（在“应用程序/实用工具”中），在右上角搜索 `coreaudiod`，选中它，点左上角“×”强制退出。这个进程会自动重启，重置音频系统，能解决大多数软性问题。
4. 若以上无效，重启 Mac 是最简单的测试手段。如果问题反复出现，可能涉及硬件，需报 IT。
