# 无线网络

## 使用指南

### 如何连接公司 Wi‑Fi

#### Windows 11 连接 WiFi 步骤

1. 打开网络列表
点击任务栏右下角的 网络图标（通常是一个小地球的图标或一个小电脑图标）。
2. 选择 WiFi
在弹出的快捷面板中，找到并点击 “管理 Wi-Fi 连接”（一个小箭头或“&gt;”符号），或者直接点击 WiFi 图标 右侧的 “&gt;” 按钮。
3. 选择要连接的网络
会列出附近可用的 WiFi 名称（SSID）。找到你公司的 WiFi 名称，点击它。
4. 连接并输入密码
点击 “连接”，然后输入 WiFi 密码。如果需要，勾选 “自动连接” 方便下次自动连上。
5. 确认安全设置
一般情况下 Windows 会自动识别加密类型，直接点下一步即可。如果弹出“是否允许你的电脑被其他设备发现”，视公司策略选择“是”或“否”。
6. 连接成功
网络图标会变成实心的信号图标，表示已连接。

#### Windows 10 连接 WiFi 步骤

1. 点击任务栏右下角网络图标（通常是一个小地球的图标或一个小电脑图标）。
    <img src="/assets/OAi8bc2JzoAg7WxEsu9c8YLCndc.png" src-width="93" src-height="40"/>

2. 选择 WiFi 网络：在弹出的面板中，会显示所有可用的无线网络，找到公司的 WiFi 名称，左键单击。
    <img src="/assets/IfPKbVoUXoLqMjxrMHjcx7nfn5f.png" src-width="308" src-height="181"/>

3. 勾选“自动连接”，左键单击 “连接” 按钮。
    <img src="/assets/EzeGbCJEhoxz13xNg9pcpM8VnQb.png" src-width="355" src-height="136"/>

4. 输入密码：在密码框中输入公司 WiFi 的密码，可以勾选 “自动连接”，然后点击 “下一步”。
    <img src="/assets/BgtMbptiXo7rLzxotSecZhninwf.png" src-width="346" src-height="167"/>

5. 完成连接：验证密码正确后，状态变为“已连接，安全”，表示WiFi已连接。
    <img src="/assets/Vz1cbNWL5otjSYx6rUzcBACun0e.png" src-width="353" src-height="138"/>

### 忘记网络重连

设置 → 网络和 Internet → WLAN → 管理已知网络 → 找到公司网络 → 忘记，然后重新连接。

## 应急上网

### 如何使用手机热点供电脑上网

手机打开个人热点，电脑搜索 Wi‑Fi 连接。

### 如何使用手机USB共享网络给电脑

数据线连电脑，手机设置中打开“USB 网络共享”，Windows 会自动识别为一个有线连接。
