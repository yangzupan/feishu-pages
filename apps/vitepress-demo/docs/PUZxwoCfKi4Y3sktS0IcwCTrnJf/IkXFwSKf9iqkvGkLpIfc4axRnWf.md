# 意见反馈

《电脑与网络使用指南》是为公司全体同事服务的文档，我们真诚希望能越来越好。如果您在使用本手册或日常办公中，发现了以下情况，欢迎随时告诉我们：

- 手册内容有误：步骤与实际情况不符、命令已失效、路径写法有错等。
- 手册覆盖不足：您遇到的某个常见问题，手册里完全没有提到。
- 操作不够清晰：您按照手册步骤操作，但看不懂、做不通，最终还是要找 IT 才能解决。
- 新问题新需求：公司上线了新系统、新设备，或出现了手册未覆盖的新型故障。
- 任何改进想法：您觉得有些内容用表格、截图或视频方式展示会更好，或者有更好的排查思路。

### 钉钉（推荐）

钉钉内搜索【杨祖攀】或者【星辰】，直接私信我。

### 表单

暂不支持。

### 当面沟通

如果您觉得线上沟通不够高效的话，可以直接线下当面沟通。

## 反馈时请尽量包含

您提供的信息越完整，我们越能快速定位并改进：

1. 您的姓名与部门（非强制，但方便我们若需深入了解时联系您）。
2. 反馈类型：内容错误 / 覆盖不足 / 操作不清 / 新需求 / 其他。
3. 具体位置：请注明涉及的手册章节、小节标题，或者直接在邮件中附上相关页码或截图。
4. 问题描述：如果反馈某个方法的错误或不通，请描述您实际遇到的状况（如：“按第 X 页的打印机清理方法操作后，依然显示脱机，后来 IT 帮我做了 XX 才解决”）。
5. 建议方案（若有）：您觉得怎样写会更清楚、怎样排布更方便查阅，欢迎直接告诉我们。

## 您的反馈将被如何对待

1. 每一条反馈都会被我们认真阅读和记录。
2. 经核实后，我们会在手册更新中修正错误、补充内容或优化表达。
3. 对于贡献了重要改进建议的同事，我们将在手册更新日志中致谢（征得本人同意后）。
