# 常用快捷键

> <b>使用说明</b>：`Win` 键指键盘上 Windows 徽标键（通常位于 `Ctrl` 和 `Alt` 之间）。`+` 表示同时按下。

> Windows 投屏模式：
> 1. 仅电脑屏幕：只在自己笔记本上显示
> 2. 复制：外接屏幕和笔记本显示相同内容
> 3. 扩展：外接屏幕作为第二块屏幕，可拖拽窗口过去
> 4. 仅第二屏幕：笔记本黑屏，只在外接屏幕显示
