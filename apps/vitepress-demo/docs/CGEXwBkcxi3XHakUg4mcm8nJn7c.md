# 前言与概述

欢迎使用《电脑与网络使用指南》！本手册是专为公司全体同事打造的“零基础、场景化”自服务工具书，其目的是为公司全体同事提供一个清晰、易用的电脑与网络使用指南。无论是<b>日常操作</b>，还是遇到<b>突发故障</b>，您都可以优先查阅本手册尝试解决，这能最大程度地节省您的时间，快速恢复工作。

1. <b>指导规范操作</b>：提供安全、合规的软件安装、账号管理、数据处理等操作指引，避免因误操作引发安全风险。

3. <b>提升工作效率</b>：介绍了常用的快捷键、系统设置和办公技巧，能够帮您更快、更顺手地完成工作。
4. <b>明确安全红线</b>：详细列出禁止事项和威胁识别方法，保护个人和公司数据免受钓鱼、勒索等攻击。

## 如何使用手册

1. <b>按目录定位</b>：当自己的电脑设备发生故障时，首先根据故障现象判断属于“电脑、设备、软件、网络、安全”中的哪一类，然后通过目录或搜索关键词寻找对应解决方案。
2. <b>阅读“使用指南”</b>：如果您是不熟悉某个功能的操作，请优先阅读该章节的“使用指南”部分，我们会用最简单的步骤教您完成相关操作。
3. <b>排查“常见故障”</b>：如果是设备出现异常，请直接查阅“常见故障”小节，并严格按照给出的步骤顺序操作。大部分问题通过<b>重启</b>、<b>检查连线</b>、<b>重装驱动</b>等基础操作即可恢复。
4. <b>先自助再报修</b>：若严格按手册指引操作后问题仍未解决，请参考附录的[《报修指南》](/IaREwRuvGikmX7kK3GxcfTLungf) 直接联系我们，我们随时为您提供支持。
5. <b>注意提示标识</b>：
    - ⚠️ <b>注意</b>：可能导致数据丢失或操作不可逆的步骤，请务必先备份，在进行操作。
    - 📌 <b>重要</b>：操作前需注意的前提条件或解释。
    - ❌ <b>禁止</b>：违反公司安全规定的行为，可能带来违规后果。

## 设备使用建议

1. 专人专用：所配电脑及外设由指定使用人负责。若同事之间确需临时交换使用设备，请务必通过交接，但不应长期借用他人账号。
2. 文件管理规范：所有工作文档、数据请尽量保存在 D 盘（若系统已分区）。不要在桌面或 C 盘长期存放重要文件，因为一旦系统崩溃需要重装，这些文件将极有可能永久丢失。
3. 备份意识：凡是经过大量劳动产生的成果，务必遵循“3-2-1”原则（3 份拷贝，2 种介质，1 份异地）进行定期备份。
4. 养成良好习惯：定时重启、及时更新、离开锁屏、不点可疑链接，能预防 80% 的故障。
